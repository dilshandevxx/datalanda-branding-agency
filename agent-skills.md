# Agent Skills: IT Service Agency Website

## Architecture & Tech Stack
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS Modules / Global CSS (No Tailwind)
- **Package Manager**: npm

## Design Philosophy (Minimal Vibe)
- **Colors**: Deep dark backgrounds (`#111111` or `#0a0a0a`), stark white text (`#ffffff`), and vibrant accent colors (like solid punchy red, orange, or electric blue) for images and highlights.
- **Typography**: Bold, tight sans-serif fonts (Inter, Helvetica Neue, or a Grotesk). Headings should be massive and uppercase for the hero section to mimic the "Tavern" inspiration.
- **Layout**: Clean, spacious, using a lot of negative space. Elements are horizontally centered or aligned strictly to a grid.
- **Components**:
  - `Header`: Minimalist, text-only logo on the left, clear and simple text links on the right.
  - `Hero`: Large typography, concise and powerful messaging.
  - `Images`: Editorial style, high quality, vibrant and highly contrasted with the dark theme.
